# Settings

## Overview

The settings center allows administrators to configure system behavior.

## Sections

```txt
General System
Default Route
Users
Security
Language
Datasheet API
Printer & Label
Appearance
Backup
Detailed Logs
Updates
```

## Language

The language setting controls the interface language.

Supported languages:

```txt
Türkçe
English
```

## Default Route

Users can choose the page opened after login:

```txt
/dashboard
/components
/projects
/low-stock
/purchase
/labels
/activity
/settings
```

Stored key:

```txt
reelmanager_default_route
```

## Appearance

Supported appearance options:

```txt
Theme
Font size
Radius
Density
Panel style
Accent color
```

## Datasheet API

Configurable provider settings:

```txt
Nexar Client ID
Nexar Client Secret
DigiKey Client ID
DigiKey Client Secret
DigiKey Redirect URI
Mouser API Key
Provider timeout
Cache TTL
Provider priority order
```

## Backup

Backup section supports:

- JSON export
- SQL export
- FTP/SFTP configuration test
- Remote path validation

## Security

Security settings may include:

- Session policy
- Auto logout time
- Password policy
- MFA option
- Rate limit
- Audit level

## Users

Administrators can manage users, roles and status.
