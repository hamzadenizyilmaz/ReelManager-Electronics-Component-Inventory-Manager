# ReelManager Documentation

Welcome to the official documentation directory for **ReelManager v2.2.0 Enterprise**.

ReelManager is an open-source electronic component inventory management platform built for SMD component tracking, stock movement control, BOM/project management, datasheet enrichment, label printing, backup workflows, audit logs and GitHub-based release management.

## Supported Stack

- Frontend: Next.js 15, React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MySQL / MariaDB
- ORM: Prisma
- Authentication: JWT
- API Docs: Swagger/OpenAPI
- CI/CD: GitHub Actions
- Process Manager: PM2
- Reverse Proxy: Nginx
